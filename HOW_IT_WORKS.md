# Как работает проект

## Общая идея

Это веб-приложение для просмотра и управления финансовыми данными. Оно показывает расходы, транзакции, графики, и позволяет «оплатить» через имитацию PayPal. Работает в браузере как одностраничное приложение (SPA).

## Как устроено

### 1. Vue 3
Vue — это фреймворк для создания интерфейсов. Каждый элемент на странице (кнопка, таблица, форма) — это компонент. Компоненты можно переиспользовать и комбинировать как кубики конструктора.

### 2. Pinia (хранилище состояния)
Когда данные нужны нескольким компонентам одновременно (например, список расходов), они хранятся в Pinia — это центральное хранилище. Любой компонент может читать и изменять данные в нём.

### 3. Vue Router
Позволяет переключать «страницы» без перезагрузки. Когда вы нажимаете на ссылку «Расходы» или «Транзакции», URL меняется, но страница не перезагружается — просто показывается другой компонент.

### 4. Взаимодействие с API
Приложение отправляет HTTP-запросы к backend (из задач 1/2/3/6) через Axios. Например: загрузить список расходов, создать новый, оплатить через PayPal.

### 5. Графики
Используется Chart.js для отображения графиков расходов по категориям, платежей по месяцам. Данные для графиков приходят с backend API.

### 6. Мок PayPal
Кнопка "Pay with PayPal" имитирует процесс оплаты:
1. Нажатие на кнопку → запрос к API `/payments/paypal/create`
2. Перенаправление на «страницу PayPal» (симуляция)
3. Подтверждение → запрос к `/payments/paypal/capture`
4. Статус расхода обновляется на «paid»

### 7. Интернационализация (i18n)
Приложение поддерживает несколько языков. Все тексты (кнопки, надписи, сообщения) хранятся в файлах переводов, и можно переключить язык одним нажатием.

### 8. Логирование ошибок
Когда в браузере происходит ошибка (например, API не отвечает), она перехватывается и отправляется на backend для записи в логи. Это помогает находить проблемы.

### 9. Docker и Nginx
Для production приложение собирается в статические файлы (HTML, CSS, JS) и раздаётся через Nginx — быстрый веб-сервер.

## Поток данных

```
Пользователь → Браузер (Vue 3)
                   ↓
              Vue Router → Страница (View)
                   ↓
              Pinia Store → Axios → Backend API
                   ↓
              Компоненты обновляются реактивно
                   ↓
              Графики, таблицы, формы
```
